## Getting Started
Planet Faucet is a JavaScript project. 
JavaScript development tools requirements:
- [Node.js] 16.13.1+
- [npm] 8.1.2+

## Project Structure
- `/views`: Implements the front-end HTML pages of the project


### Prerequisites

### 1. Install Dependencies
>npm  install


### 2. Run the project
>npm run serve

or
>node server.js
